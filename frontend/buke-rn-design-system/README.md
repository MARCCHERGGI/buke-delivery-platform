# Buke React Native Design System

Fast, map-friendly, one-handed mobile design system for Bukë.

## Philosophy

- Immediate comprehension over decoration.
- Big tap targets and compressed decision depth.
- Strong hierarchy for ETA, price, and order progress.
- Bright surfaces, dark text, and a single green brand accent.

## Included

- Color, typography, spacing, radius, and shadow tokens
- `RestaurantCard`
- `MenuItemCard`
- `CartPanel`
- `OrderTimeline`
- `DriverCard`
- `RatingBadge`
- `CategoryScroller`
- `SearchBar`
- `DeliveryETAWidget`
